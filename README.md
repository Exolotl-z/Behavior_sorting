# 鼠行为标记系统

行为视频手动分析工具，做了打包

## ⚡ 快速开始

### 方法1: 直接运行（需要Python）
1. 双击 `install_dependencies.bat` 安装依赖
2. 双击 `运行程序.bat` 启动程序

### 方法2: 使用exe文件（推荐）
1. 双击 `build_exe.bat` 打包程序
2. 运行 `dist/MouseBehaviorAnnotator.exe`

## ✨ 主要功能

- 🎬 **视频播放与控制** - 支持多种格式，流畅播放
- 🎯 **实时行为标记** - 快捷键标记，时间线可视化
- 📊 **数据可视化分析** - 统计图表，直观展示
- 💾 **自动数据导出** - CSV格式，便于进一步分析
- 🖥️ **友好用户界面** - 现代化设计，操作简单

## 📖 详细文档

请查看 **使用说明.md** 获取完整使用教程

## 🎮 快捷键

- **G** - 标记行为开始/结束
- **空格** - 暂停/继续
- **J/L** - 后退/前进 5秒
- **C** - 保存数据
- **Q** - 退出程序

1. **选择视频** → 2. **设置分析窗口** → 3. **标记行为** → 4. **查看分析** → 5. **保存数据**
